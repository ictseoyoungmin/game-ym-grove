# Ym Grove v0.1 Balance Notes

## First Unlock Route

Recommended first variant: `AI / Agents Ym`.

- Train Intelligence from 0 to 3: `20 + 25 + 30 = 75 Spark`
- Train Connection from 0 to 3: `20 + 25 + 30 = 75 Spark`
- Evolve AI / Agents Ym: `180 Spark`
- Total expected Spark: `330`

With active tapping at roughly one tap per second for five minutes, the player earns about `300 Spark` from taps and `30 Spark` from Core Ym idle production. This puts the first unlock at about five minutes without requiring offline waiting.

## Early Progression

- Core Ym provides the base `0.10 Spark/sec`.
- AI / Agents adds a noticeable `0.25 Spark/sec` after the first unlock.
- ML / Deep Learning and JEPA / Vision remain reachable by continuing Intelligence or adding Curiosity.
- Insight becomes meaningful through `Reveal Hint`, which costs `5 Insight` and persists in the save.

## v0.1 Policy

Trust is a stability reserve for later systems. v0.1 keeps evolution deterministic and uses Trust as a visible local resource without adding failure probability.
